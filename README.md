 # 🎓 Student Result Management System (SRMS)

## 📌 Overview
The Student Result Management System (SRMS) is a system designed using System Analysis and Design principles to manage student academic records efficiently.

It centralizes student data, automates result processing, and provides quick access to academic performance.

---

## 🎯 Objectives
- Centralized student record management
- Automated marks entry and grade calculation
- Secure login system for different users
- Fast and accurate result publication

---

## 👥 User Roles
- **Administrator** – Manages system, users, and results
- **Teacher** – Enters marks and generates reports
- **Student** – Views results and downloads reports

---

## ⚙️ Features
- Student information management
- Course & subject management
- Marks entry system
- Automatic grade calculation
- Result publishing
- Report generation
